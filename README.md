# 云音乐开发

**前端技术栈:**

> vue2全家桶 + ElementUI

**后端接口主要来源于：**

> GitHub开源接口https://github.com/Binaryify/NeteaseCloudMusicApi

**功能**：

- 登录后可查看自己歌单
- 在线播放音频
- 搜索歌曲
- 播放记录
- 动态歌碟
- 滚动歌词
- 排行榜
- ....

**页面**：本项目总共包含13个页面

**难点**：歌词随着歌曲平滑同步移动
